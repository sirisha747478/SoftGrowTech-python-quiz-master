# 🐍 Python Quiz Game

A command-line based quiz game developed using Python. This project focuses on testing the user's knowledge of Python programming through multiple-choice questions.

---

## 📌 Features

* 🧠 Python-related quiz questions
* 📋 Multiple-choice options (A, B, C, D)
* 🎯 User input for answers
* ✅ Instant answer validation
* 📊 Score calculation
* 🏁 Final result display

---

## 🛠️ Technologies Used

* Python 3
* Concepts used:

  * Loops (`for`, `while`)
  * Conditional statements (`if-else`)
  * Lists & Dictionaries
  * Functions

## ▶️ How to Run

1. Clone the repository

```
git clone https://github.com/your-username/python-quiz-game.git
```

2. Navigate to the project folder

```
cd python-quiz-game
```

3. Run the program

```
python quiz.py
```

---

## 🎮 How It Works

1. The program displays Python-related questions.
2. Each question has 4 options (A, B, C, D).
3. The user selects an answer.
4. The program checks if the answer is correct.
5. Score is updated based on correct answers.
6. Final score is displayed at the end.
